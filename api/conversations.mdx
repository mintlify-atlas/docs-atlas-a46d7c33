title: Conversations
description: Manage groups, DMs, and message streams
---

The `Conversations` class handles listing, creating, and streaming conversations. It provides methods for both group chats and direct messages.

## Overview

Access via the `client.conversations` property:

```swift
let conversations = client.conversations
```

## Finding Conversations

### findGroup

Find a specific group by ID.

```swift
func findGroup(groupId: String) throws -> Group?
```

<ParamField path="groupId" type="String" required>
  The hex-encoded group ID
</ParamField>

<ResponseField name="result" type="Group?">
  The group if found, nil otherwise
</ResponseField>

**Example:**

```swift
if let group = try conversations.findGroup(groupId: "a1b2c3...") {
    print("Found group: \(try group.name())")
}
```

## Listing Conversations

### listGroups

List group conversations.

```swift
func listGroups(
    createdAfterNs: Int64? = nil,
    createdBeforeNs: Int64? = nil,
    lastActivityAfterNs: Int64? = nil,
    lastActivityBeforeNs: Int64? = nil,
    limit: Int? = nil,
    consentStates: [ConsentState]? = nil,
    orderBy: ConversationsOrderBy = .lastActivity
) throws -> [Group]
```

<ParamField path="createdAfterNs" type="Int64">
  Filter groups created after this timestamp (nanoseconds)
</ParamField>

<ParamField path="createdBeforeNs" type="Int64">
  Filter groups created before this timestamp (nanoseconds)
</ParamField>

<ParamField path="lastActivityAfterNs" type="Int64">
  Filter groups with activity after this timestamp
</ParamField>

<ParamField path="lastActivityBeforeNs" type="Int64">
  Filter groups with activity before this timestamp
</ParamField>

<ParamField path="limit" type="Int">
  Maximum number of groups to return
</ParamField>

<ParamField path="consentStates" type="[ConsentState]">
  Filter by consent states
</ParamField>

<ParamField path="orderBy" type="ConversationsOrderBy">
  Sort order: `.createdAt` or `.lastActivity` (default)
</ParamField>

<ResponseField name="groups" type="[Group]">
  Array of group conversations
</ResponseField>

**Example:**

```swift
let recentGroups = try conversations.listGroups(
    limit: 20,
    orderBy: .lastActivity
)
```

### newGroup

## Streaming

### stream

Stream new conversations as they are created.

```swift
func stream(
    type: ConversationFilterType = .all,
    onClose: (() -> Void)? = nil
) -> AsyncThrowingStream<Conversation, Error>
```

<ParamField path="type" type="ConversationFilterType">
  Filter by type: `.all`, `.groups`, or `.dms` (default: `.all`)
</ParamField>

<ParamField path="onClose" type="(() -> Void)?">
  Optional callback when stream closes
</ParamField>

<ResponseField name="stream" type="AsyncThrowingStream<Conversation, Error>">
  Stream of new conversations
</ResponseField>

**Example:**

```swift
for try await conversation in conversations.stream() {
    print("New conversation: \(conversation.id)")
}
```

### streamAllMessages

Stream all messages from all conversations.

```swift
func streamAllMessages(
    type: ConversationFilterType = .all,
    consentStates: [ConsentState]? = nil,
    onClose: (() -> Void)? = nil
) -> AsyncThrowingStream<DecodedMessage, Error>
```

<ParamField path="type" type="ConversationFilterType">
  Filter by conversation type: `.all`, `.groups`, or `.dms` (default: `.all`)
</ParamField>

<ParamField path="consentStates" type="[ConsentState]">
  Optional filter by consent states
</ParamField>

<ParamField path="onClose" type="(() -> Void)?">
  Optional callback when stream closes
</ParamField>

<ResponseField name="stream" type="AsyncThrowingStream<DecodedMessage, Error>">
  Stream of messages from all conversations
</ResponseField>

**Example:**

```swift
for try await message in conversations.streamAllMessages(type: .groups) {
    print("New message: \(message.id)")
}
```
