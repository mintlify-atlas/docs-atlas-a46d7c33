    Get your iOS app sending secure messages in minutes
  </p>

  <Steps>
    <Step title="Install the SDK">
      Add the XMTP iOS SDK to your project using Swift Package Manager.

      ```swift
      dependencies: [
        .package(url: "https://github.com/xmtp/xmtp-ios", from: "4.0.0")
      ]
      ```

      See the [installation guide](/installation) for CocoaPods and detailed setup instructions.
    </Step>

    <Step title="Create a client">
      Initialize an XMTP client with a signing key to authenticate your user.

      ```swift
      import XMTPiOS

      let client = try await Client.create(
        account: account,
        options: .init(
          api: .init(env: .production)
        )
      )
      ```
    </Step>

    <Step title="Send your first message">
      Start a conversation and send an encrypted message.

      ```swift
      let conversation = try await client.conversations.newConversation(
        with: "0x1234..."
      )
      
      try await conversation.send(content: "Hello from XMTP!")
      ```

      Check out the [quickstart guide](/quickstart) for a complete working example.
    </Step>
  </Steps>
</div>
